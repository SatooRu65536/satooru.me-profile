デザインは旧バージョンと同じ

- panda css (CSSinJS) を触れた
- Next14(AppRouter) と Static Export (静的レンダリング) を使った
- コンポーネントの分け方を意識した
    - 汎用性が高くなるようにした
    - テストが書きやすい構造にした
- AppRouter に合わせたアーキテクチャにした
- ESLint と Prettier で統一的な書き方になるようにした
- issuesを活用してブランチを使い分けるようにした
- etc.
