# 概要

Flutter が分からず挫折したアプリです

```
海に流して知らない人にメッセージが届くボトルメール、のアプリ版です。
流した瓶はマップ上を彷徨ってどこか知らない地へ届きます。
近くに流れ着いたら拾ってみよう。

どこの誰に届くか分からないメッセージ。
だれかに言うまでもない呟きを流そう。
```

# 技術スタック

- Flutter
- firebase
